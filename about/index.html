Mahjong Pantheon

Mahjong Pantheon is a set of web-based software tools to automate regular riichi mahjong tournaments & local games.

Subsystems

  • Mimir is a specialized backend database which supports JSON-RPC calls for data operations.
  • Frey is a backend for authorization and personal data management.
  • Tyr is mobile web application, which uses Mimir as its backend to provide online game overview and automated scoring abilities for every player with handheld device. See screenshots.
  • Rheda is simple dynamic web frontend to show Mimirâ€™s data pretty and conveniently. It includes rating tables, statistics, graphs, last games and also some tournament administration tools. See screenshots.

Software stack

All Pantheon tools are supposed to be web-based applications, which run either in browser or on a server. Tyr is written almost entirely in Typescript, Mimir, Frey and Rheda are written in PHP (7.0+). Mimir and Frey use PostgreSQL database as its backend.

Production installation instructions are still WIP, but it should be pretty easy to install all four subsystems for anyone who is familiar with php-related web development. Source code and development environment setup instructions are available on github.
